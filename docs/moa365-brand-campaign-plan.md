## 6. 운영 원칙 리마인드 (전 Phase 공통)

1. 퍼포먼스 소재에 '여정/항해/선장' 금지 — '여정'은 L2 이후 의미 부여 문맥에서만.
2. 모든 소재에 점-선 모티프 1회 이상 + 4색 시맨틱.
3. AI 생성 인물 금지 — 실제 앱 화면·실제 사용 장면·손맛 요소 우선. (Seedance 실사풍 인물 클립 2종은 §7-4 위반으로 **사용 보류**)
4. 푸시·빈 상태·챌린지 커뮤니케이션에 죄책감 유발 문구 금지 — 항상 문턱을 낮추는 방향.
5. 모든 게시물 UTM 분리, 공유 카드는 전용 딥링크.
