## ★ SAMPLE #1 — 무자막 3초 데모 쇼츠 (Ver.C) **[가장 먼저 제작]**

**왜 첫 번째인가**: 제작비 최소·설치효율 최대(§4 #2). L1 100%(메타포 0). 퍼널 입구(인지→방문→설치) 직접 개방. 브랜드 필름(아우라)은 퍼널 뒤(§4 #5).

| 항목 | 값 |
|---|---|
| 포맷 | 세로 9:16, 1080×1920 |
| 길이 | **3.0s** (핵심) / 확장본 6s·9s 파생 |
| 자막 | **무자막**(텍스트 1개만 — 마지막 0.5s 로고락업) |
| 톤 | 실사 화면 녹화(앱 실제 동작) — AI 스멜 금지(§7-4) |
| 사운드 | 타이핑 1회 + 완료 "틱" SFX, 보이스오버 없음 |
| 레이어 | **L1 100%** — 메타포·세계관 단어 0 |

### 컷 시트 (3초)

| 시간 | 화면 | 동작 | 브랜드 요소 |
|---|---|---|---|
| 0.0–1.0s | 채팅 입력창 클로즈업 | 손가락이 **"스벅 6500"** 타이핑 | 입력 캐럿 = **Record 오렌지** |
| 1.0–2.0s | 엔터 → AI 파싱 | 한 줄이 **카테고리·금액·날짜 카드**로 변환(카페 ☕ / -6,500 / 오늘) | 카드 보더 = Record→Journey 그라데이션 |
| 2.0–2.5s | 카드가 캘린더의 한 점으로 안착 | 점 하나가 **선** 위에 찍힘 | **점-선 모티프**(점=오늘 기록) — §7-3 충족 |
| 2.5–3.0s | 로고 락업 | 점선 + 로고 "moa365" + 단 한 줄 카피 | 하단 워터마크(공유=광고 구조) |

### 온스크린 텍스트 (단 1개, 2.5s 등장)
```
말하듯 한 줄, 가계부 끝.
```
> 히어로 슬로건과 동일(`landing.headline1/2`) — 카피 일관성. 영문판: **"One line a day."**

### 캡션/해시태그 (게시용, 채널별)
- **공통 본문**: `"스벅 6500" 한 줄이면 끝. 카테고리 선택도, 계좌 연동도 없이.`
- IG/TikTok: `#하루한줄 #가계부 #가계부어플 #AI가계부 #moa365`
- YouTube Shorts: 제목 `"스벅 6500" 한 줄이면 가계부 끝 #Shorts`

### 검수 게이트 통과 (§7)
1. 행동 지시 = 평어("한 줄이면 끝") ✅ 메타포 0
2. 의미 환원 없음(퍼포먼스 소재 L1 100%) ✅
3. 점-선 모티프 1회(2.0–2.5s) ✅
4. AI 스멜 없음 — 실제 앱 화면 녹화, 가짜 인물·후기 0 ✅
5. 퍼포먼스 소재이므로 1·2 엄격 적용(서사는 브랜드 소재에서) ✅

### 제작 옵션
- **A안(권장·정확)**: 실제 앱 화면 스크린 레코딩(에뮬레이터/실기기) → 컷 편집. 손맛·진정성 최상, 비용 0.
- **B안(빠른 프로토)**: 키프레임 합성 영상으로 시안 먼저 → 승인 후 A안 정식. (영상 생성 도구 사용 — 크레딧 소요)

### 파생물(같은 소스에서)
- Ver.A 공감형(6s): 영수증 쌓임 → 한 줄로 정리되는 안도. **L2** 진입.
- Ver.B 대립형(6s): "자동연동의 정체불명 내역" vs "내가 쓴 내 한 줄". **포지셔닝**.
