## Part 0. 이 문서의 사용법

- **내부 세계관 언어**(항해·선장·항해일지)는 기획·디자인·콘텐츠 제작의 '관점'으로만 사용한다. 유저에게 행동을 지시하는 모든 표면(온보딩, 버튼, 스토어, 퍼포먼스 광고)에는 절대 노출하지 않는다.
- **표면 언어 원칙: 행동은 평어로, 의미는 메타포로.** 메타포는 입력(시키는 말)이 아니라 출력(의미를 돌려주는 말)에만 쓴다.
- 표면에 허용되는 세계관 단어는 단 하나, **'여정'**.

## Part 1. 브랜드 전략 기반 (Brand Core)

### 1-1. 세계관 (내부 비공개 — 모든 의사결정의 북극성)

> 누구나 자기만의 항로를 가진 선장이다.
> 그런데 대부분, 항해일지 없이 항해한다.
> 가늠으로 가는 배는 표류한다.
> 대충 가늠하는 삶은, 대충 계획하는 삶이다.

### 1-2. 업의 본질

- ~~대화형 AI 가계부 앱~~ (시장 천장 = 가계부 앱)
- → **인생 여정의 재무 기록·항법 시스템** (시장 천장 = 인생 설계)
- 월간 AI 코칭, 목표 설계, 연간 회고는 부가기능이 아니라 본질의 일부다.

### 1-3. 역할 구도 (경쟁 대립각)

| 주체 | 역할 | 핵심 |
|---|---|---|
| 유저 | 선장 | 키를 잡는 주체. 한 줄 기록 = 키를 잡는 최소 행위 |
| moa365 | 항해일지 + 항해사 | 기록을 받아 항로를 짚어주되, 키는 절대 잡지 않는다 |
| 토스/뱅크샐러드 | 자동 운항 장치 | 편하지만 내가 항해하지 않는 배. 판단의 '위탁' |

**철학적 차별점 한 문장:** 자동으로 쌓이는 데이터는 에피소드, 내 손으로 쓴 한 줄은 서사. 아우라(브랜드 자산)는 서사에만 쌓인다.

### 1-4. 타겟 정의 (이중 레이어)

- **획득 언어(광고용·결핍 프레임):** "가계부, 또 포기하셨죠?" — 가계부 작심삼일러, 토스 알림만 쌓아두는 사람
- **브랜드 언어(서사 프레임):** 자기 인생의 여정을 가진 사람. 결핍으로 데려와서, 선장으로 만들어준다.
- 유저 호칭(커뮤니티/CRM): **"선장님"** — 단, 가입 후 D30 이상 활성 유저 대상 커뮤니케이션부터 점진 도입. 첫 대면에는 사용 금지.

### 1-5. 언어 3레이어 체계

| 레이어 | 시점 | 언어 | 예시 |
|---|---|---|---|
| L1 첫 대면 | D0 | 메타포 0%, 기능 평어 | "말하듯 한 줄, 가계부 끝." |
| L2 사용 중 | D1~D90 | 축적 평어 | "기록 30일째. 소비 패턴이 보이기 시작했어요." |
| L3 의미 부여 | 월말·연말·브랜드 콘텐츠 | 세계관 언어 허용 | "당신이 직접 쓴 365개의 한 줄로 만든, 당신의 1년." |

### 1-6. 브랜드 보캐뷸러리 맵 (내부용 ↔ 표면용 대조표)

| 기능 | 내부 세계관 언어 | 표면 노출 언어 |
|---|---|---|
| 한 줄 입력 | 오늘의 항해일지 | 오늘 한 줄 |
| 지출 캘린더 | 항적(航跡) | 기록 캘린더 |
| 잔액 요약 | 현재 위치와 연료 | 이번 달 요약 |
| 월간 AI 코칭 | 항해사의 항로 브리핑 | 월간 리뷰 |

## Part 4. 키 비주얼 (Key Visual)

### 4-1. 핵심 모티프: **"한 줄 → 길"**

- 채팅 입력창에 적힌 한 줄 텍스트의 **밑줄(baseline)이 화면 밖으로 연장되어 지평선/항로가 되는** 그래픽.
- 점(dot) = 하루의 기록, 선(line) = 여정. 365개의 점이 이어져 하나의 길을 이룬다.
- 이 모티프 하나로 L1(기능: 한 줄 입력)과 L3(의미: 여정)을 한 화면에 동시 표현 — 언어로 설명하지 않고 시각으로 세계관을 전달한다.

### 4-2. 디자인 시스템 스펙 (레퍼런스 기반 상세)

**기본 레이아웃 해부 (16:9, 1280×720 기준)**

- 화면 상단 약 13%는 **화이트 밴드**: 좌측 상단에 섹션 라벨만 소형 볼드로 배치 (예: "Color System", "Graphics", "Key Visual"). 라벨 외 요소 없음.
- 나머지 87%는 **순수 블랙(#000000) 패널**: 모든 그래픽·타이포가 이 위에서 발광하듯 떠 있는 구조. 블랙의 여백이 곧 디자인 — 요소 간 간격을 과감하게 비운다.
- 우측 하단에 **moa365 워드마크**를 소형 자간 넓은 화이트 타입으로 고정 (m o a 3 6 5).
- 서체: 지오메트릭 산스 계열(Pretendard 권장), 컨셉 워드는 화이트 볼드 대형, 설명문은 라이트 웨이트 소형 2줄.

**A. 컬러 시스템 — 그라데이션 필(Pill) 구조**

블랙 위에 좌우로 길게 뻗는 **캡슐(알약)형 그라데이션 바** + 동일 계열 **원형 아이콘 칩**(화이트 심볼 내장)을 한 쌍으로 운용. 필 안에는 컨셉 워드 1개만 화이트 볼드로. 필이 화면 좌우 가장자리에서 잘려 나가도록 배치해 '계속 이어지는 선(여정)'의 인상을 준다. moa365 시맨틱 컬러 4종:

| 컨셉 | 의미 | 그라데이션 | 아이콘 칩 |
|---|---|---|---|
| **Record** | 한 줄 기록, 행동의 순간 | 오렌지→앰버 (#FF6B2C→#FFA63D) | ✎ 연필/입력 |
| **Journey** | 쌓여가는 여정 | 시안→블루 (#01DCE3→#2D7DFF) | → 진행 화살 |
| **Insight** | 월간 AI 리뷰, 패턴 발견 | 바이올렛→퍼플 (#7B61FF→#A78BFA) | ◉ 렌즈/달 |
| **Growth** | 목표·더 나은 삶 | 라임→그린 (#9BE15D→#00C48C) | ⚑ 깃발 |

- Record 오렌지는 ATM 브랜드 액센트(#D97757) 계열과 톤을 맞춰 모브랜드와의 연결감 유지.
- 운용 규칙: 한 화면에 필 2~3개 + 칩 2~3개를 상하 2단으로 엇갈리게 배치(지그재그 리듬). 모든 필의 그라데이션 방향은 좌→우 통일(시간이 흐르는 방향 = 여정).

**B. 그래픽 모티프 — 추상 심볼 4종 (블랙 위 단색 그라데이션 도형 + 하단 컨셉 워드 + 설명 2줄)**

| 심볼 | 형태 묘사 | 컨셉 워드 | 설명문(슬라이드 표기용) |
|---|---|---|---|
| **One Line** | 발광하는 점 1개에서 우측으로 가늘게 뻗어나가는 단일 라인. 점은 크고 선은 점점 가늘어짐 | One Line | 하루 한 줄, 가장 작은 기록 / 키를 잡는 최소한의 행위 |
| **Gather** | 점들이 격자(스월/나선 배열)로 차곡차곡 채워지는 도트 그리드. 일부는 채워지고 일부는 아웃라인(아직 안 쓴 날) | Gather | 흩어진 하루하루가 모이는 곳 / 모아, 365일의 축적 |
| **Path** | 점 7~9개가 완만한 곡선으로 연결된 경로. 마지막 점만 크게 발광('오늘') | Path | 한 줄이 이어져 길이 된다 / 직접 쓴 기록만이 남기는 항로 |
| **Lens** | 동심원이 바깥으로 확장되는 레이더/파동형 서클. 중심에 작은 점 | Lens | 쌓인 기록을 비추는 시선 / 월간 AI 리뷰가 발견하는 패턴 |

- 도형은 모두 **원과 선의 조합으로만** 구성(4-1 모티프의 변주). 컬러는 A의 시맨틱 컬러와 1:1 매핑 (One Line=Record 오렌지, Gather=Journey 블루, Path=Growth 그린, Lens=Insight 퍼플).
- 그라데이션은 도형 내부에서 라이트→딥 방향, 블랙 배경과의 경계는 소프트 글로우 최소화(플랫+미세 광택).

**C. 레이아웃 패턴 2종 (산출물 적용)**

- **Pattern A — 제품 중심 (퍼포먼스/스토어/랜딩):** 라이트 배경 예외 허용. 좌측 폰 목업(한 줄 입력 화면) + 우측으로 Record 오렌지 필이 화면 밖까지 뻗는 구성 + 기능 슬로건.
- **Pattern B — 서사 중심 (브랜드/캠페인/결산 카드):** 위 블랙 패널 시스템 그대로. Path 심볼 또는 365 도트 그리드 + 브랜드 슬로건 단독. 연말 'Moa Wrapped' 카드 5장도 이 시스템으로 통일.

### 4-3. 비주얼 운영 원칙

- 배·돛·나침반·키(舵) 등 **직접적 항해 오브제는 사용 금지.** 메타포는 '선과 점'으로만 추상화한다. (직접 묘사 = 촌스러움 + 기능 혼선의 주범)
- 점-선 모티프는 앱 아이콘, 로딩 애니메이션, 연속 기록 UI, 결산 카드까지 일관 적용해 브랜드 상기력(컨셉↔이름 연결)을 높인다.
- AI 생성 이미지 사용 시 '인물 사진' 금지(강연의 AI 스멜 경계). 그래픽·모션은 허용하되 손맛 요소(실제 손글씨 한 줄, 실제 영수증 질감)를 1개 이상 섞는다.

## Part 5. 영상 크리에이티브 방향

### 5-1. 브랜드 필름 (60~90초, 1편) — "한 줄"

- **구조:** 매니페스토 풀 버전 내레이션 + 일상 지출의 순간들.
- **씬 설계:** 김치찌개를 먹는 손 → 폰에 "점심 김치찌개 9천 원" 한 줄 → 그 글자의 밑줄이 화면을 가로지르는 선으로 자라남 → 다른 사람들의 한 줄들(버스비, 아이 학원비, 데이트, 병원비)이 각자의 선이 됨 → 선들이 모여 밤하늘의 항로처럼 펼쳐짐 → "한 줄이 모여, 여정이 됩니다."
- **제작 노트:** 지출 장면은 실사(스마트폰 촬영 질감 OK), 선 그래픽만 모션. 진정성 = 실사, 세계관 = 그래픽으로 역할 분리.
- **활용:** YouTube 채널 고정, 랜딩 히어로 영상, About 페이지.

### 5-2. 퍼포먼스 숏폼 (15~30초, 3종 A/B) — 전부 L1 언어만 사용

- **Ver.A 공감 후킹:** "가계부 어플 깔았다 지운 횟수 3회 이상이면 손" → 데모(말하듯 한 줄 입력 → 자동 분류) → "말하듯 한 줄, 가계부 끝."
- **Ver.B 대립각:** "토스 지출내역 한 달 전 거 기억나세요?" → "자동 기록은 당신 기억이 아니에요" → 데모 → CTA
- **Ver.C 순수 데모:** 무자막 3초 안에 입력→완료 화면. "이게 끝이에요." (앱 설치 캠페인 최적화용)

### 5-3. 빌드 저니 콘텐츠 (ATM 채널 — 진행 중인 e-book/시리즈와 통합)

- 강연의 핵심 통찰 적용: **만든 사람의 고통의 서사가 곧 브랜드의 아우라다.** moa365의 차별화 자산은 기능이 아니라 "마케터가 AI로 직접 만든 앱"이라는 반복 불가능한 서사.
- 시리즈명 제안: **"앱 하나 만드는 데 진심인 마케터"** — 실패 로그(레일웨이→렌더 이전, 결제 연동 삽질)를 숨기지 않고 공개. 에러와 결함의 기록이 아우라를 만든다.
- 각 화 엔딩 고정 멘트: "오늘도 한 줄 남겼다." (브랜드 행위와 제작 서사의 동기화)

## Part 7. 서비스 개선 지침 — Layer 1 (첫 대면)

### 7-1. 온보딩 3화면 실제 문안

**화면 1 (가치 제안)**
- 헤드: **말하듯 한 줄, 가계부 끝.**
- 서브: 카테고리 선택도, 계좌 연동도 없어요. 그냥 쓰세요.
- 비주얼: 채팅 입력창에 타이핑 애니메이션 "점심 김치찌개 9천원"

**화면 2 (작동 방식)**
- 헤드: **AI가 알아서 정리해요.**
- 서브: 날짜, 금액, 카테고리 — 한 줄에서 전부 읽어냅니다.
- 비주얼: 한 줄 → 캘린더에 자동 배치되는 모션

**화면 3 (첫 행동 유도)**
- 헤드: **오늘 쓴 돈, 하나만 적어볼까요?**
- 서브: 예시처럼 편하게요. "커피 4500" / "버스 1500" / "치킨 23000"
- CTA 버튼: **첫 줄 쓰기**
- 입력 완료 직후 피드백: "끝! 이게 다예요. 내일 또 한 줄이면 충분해요."

### 7-2. Google Play 스토어 카피

- **앱 이름:** moa365 - 한 줄로 끝나는 AI 가계부
- **짧은 설명(80자):** 말하듯 한 줄만 쓰세요. AI가 정리하는 대화형 가계부, 월간 리포트까지.
- **긴 설명 도입부:**
  가계부, 또 포기하셨나요?
  카테고리 고르다 지치고, 연동하다 귀찮아지고.
  moa365는 다릅니다. "점심 9천원" — 이 한 줄이면 끝이에요.
  (이하 기능 불릿: 한 줄 입력 / 자동 분류 캘린더 / 이번 달 요약 / 월간 AI 리뷰 / 프리미엄 안내)
- **스크린샷 캡션 순서:** ①한 줄 입력 ②자동 캘린더 ③월간 요약 ④AI 월간 리뷰 ⑤나의 1년
- 금지: 스토어 내 '항해/선장/여정' 사용 금지. '여정'은 마지막 스크린샷(나의 1년) 캡션에서만 1회 허용.

### 7-3. 빈 화면(Empty State) & 푸시 문안

- 첫 기록 전 홈: "아직 한 줄도 없어요. 오늘 쓴 돈 하나만 적어볼까요?"
- D1 푸시(저녁 9시): "오늘 한 줄, 썼어요? 10초면 돼요."
- D3 미접속 푸시: "3일치를 몰아 써도 괜찮아요. 기억나는 것만, 한 줄씩."
- 푸시 원칙: 죄책감 유발 금지("벌써 며칠째 안 쓰셨네요" 류 금지). 항상 문턱을 낮추는 방향.

## Part 8. 서비스 개선 지침 — Layer 2·3 (축적과 의미)

### 8-1. L2 축적 피드백

- 연속 기록 카운터: "기록 N일째" (홈 상단, 점-선 그래픽으로 시각화 — 키 비주얼과 동일 모티프)
- 7일 달성: "일주일을 직접 기록했어요. 자동 기록은 못 주는 거예요."
- 30일 달성: "30개의 한 줄. 당신의 한 달이 문장으로 남았어요." + 30일 카드 생성 제안
- 끊김 복귀 시: "다시 돌아왔네요. 끊긴 건 기록이지, 여정이 아니에요." (L2.5 — 세계관 단어 '여정' 첫 허용 지점)

### 8-2. 월간 AI 리뷰 포맷 (프리미엄 핵심 = 전환 후킹 포인트)

고정 4단 구조:

1. **이번 달 한눈에:** 총지출, 기록 일수, 전월 대비. (숫자, 평어)
2. **패턴 발견:** AI가 한 줄들에서 읽어낸 행동 패턴 1~2개. 반드시 유저가 쓴 문장을 인용. 예: "'스트레스 야식'이라는 단어가 이번 달 4번 등장했어요. 모두 수요일이네요."
3. **다음 달 한 가지:** 제안은 딱 1개. (조언 과잉 금지 — 항해사는 브리핑하되 키를 잡지 않는다)
4. **마지막 한 줄(L3 — 세계관 허용 구간):** "이번 달도 당신의 여정에 31줄이 더해졌어요."

- 무료 유저에게는 1번만 노출 + 2~4번 블러 처리: "당신의 한 줄들이 어떤 이야기를 만들고 있는지, 월간 리뷰에서 확인하세요." → 프리미엄 전환 동선.

### 8-3. 연말 결산 "나의 1년" (Moa Wrapped) 포맷

공유 가능한 카드 시퀀스(5장):

1. 올해 당신이 쓴 한 줄: **N개**
2. 가장 많이 등장한 단어 TOP3 (예: 커피 87회 / 택시 41회 / 치킨 32회)
3. 당신이 쓴 문장 중 AI가 뽑은 올해의 한 줄 (예: 3월 14일 "엄마 생신 케이크 38000")
4. 365개의 점으로 그린 당신의 1년 항로 그래픽 (키 비주얼 Pattern B)
5. 엔딩 카드: "한 줄이 모여, 여정이 됐습니다. — 2026년의 (닉네임)님"

- 모든 카드 하단 워터마크: moa365 로고 + 점-선 모티프 (공유 = 광고가 되는 구조)
- '1년 전 오늘' 기능: 365일 이상 유저 홈에 작게 노출. "1년 전 오늘 당신의 한 줄: ___"
